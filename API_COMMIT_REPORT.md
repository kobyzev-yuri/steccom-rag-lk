## 🎯 **Функциональность API:**

### ✅ **Реализованные эндпоинты:**
- `GET /` - Информация о сервере
- `GET /health` - Проверка состояния
- `POST /auth/login` - Аутентификация
- `GET /database/schema` - Схема БД
- `POST /sql/generate` - Генерация SQL
- `POST /sql/execute` - Выполнение SQL
- `POST /rag/query` - RAG запросы
- `GET /rag/knowledge-bases` - Список БЗ
- `GET /reports/standard` - Список стандартных отчетов
- `GET /reports/standard/{type}` - Выполнение стандартного отчета
- `GET /export/csv` - Экспорт в CSV
- `POST /charts/create` - Создание графиков
- `GET /session/state` - Получение состояния сессии
- `POST /session/state` - Сохранение состояния сессии
